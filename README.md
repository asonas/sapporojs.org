# Web site for Sapporo.js

http://sapporojs.org

## 動作確認環境

* Mac OS X Lion
* Ruby 1.9.3p194

## セットアップ

1) [jekyll]( http://jekyllrb.com/ ) をインストールする

```sh
$ gem install jekyll
```

2) ソースコードを取得して配置する

```sh
$ git clone git://github.com/sapporojs/sapporojs.org.git
```

3) サーバを起動する

```sh
$ jekyll --server
```
